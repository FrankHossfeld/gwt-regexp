# GWT Callbacks
Regular expression module for GWT 2 and J2CL.

### Dependency

```xml
<dependency>
    <groupId>org.gwtproject.regexp</groupId>
    <artifactId>gwt-regexp</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
